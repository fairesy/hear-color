# hear-color : visualize&search music using color
---
###visualizer
실시간으로 색과 모양이 변하는 비주얼라이저와 함께 음악을 플레이한다.  
###music to color
음악을 넣으면 그 음악에 해당하는-그 음악의 분위기를 표현하는- 색을 보여준다.   
###color to music
색을 선택하면 해당색을 대표색으로 갖는 음악들을 검색하여 리스트로 보여준다.
